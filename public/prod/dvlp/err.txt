概念流程圖:

```digraph G {
    程式開始 -> 關閉所有燈
    關閉所有燈 -> 建立隨機種子
    建立隨機種子 -> 重複執行以下步驟:
    建立隨機種子 -> 隨機選擇一種顏色
    隨機選擇一種顏色 -> 建立循環張數i=0,直到i=24:
    建立循環張數i=0,直到i=24 -> 將每顆燈用隨機選擇的顏色點亮
    將每顆燈用隨機選擇的顏色點亮 -> 等待0.01秒
    等待0.01秒 -> 回到建立循環張數i=0,直到i=24
    回到建立循環張數i=0,直到i=24 -> 等待0.01秒
    等待0.01秒 -> 回到建立隨機種子
}
```
